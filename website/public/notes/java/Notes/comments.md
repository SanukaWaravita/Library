```java
public class Main { 

 

public static void main(String[] args) { 

// This is a comment 

}  

} 
```

- `// This is a comment" is a comment`.  

- Anything followed by `//` is considered a comment, and the compiler will ignore these statements.  

---

```java
public class Main { 

public static void main(String[] args) { 

/* This 
* Is 
* A 
* Multi-line 
* comment 
*/ 

}  

} 
```

- Anything after `/*` and before `*/` are considered to be within the bounds of a comment and so will be treated as one.
