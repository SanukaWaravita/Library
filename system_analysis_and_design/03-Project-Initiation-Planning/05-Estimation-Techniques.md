# 05 - Estimation Techniques

## Overview

Project estimation involves predicting the effort, duration, and cost required to complete a project. Effective estimation combines historical data, expert judgment, and analytical techniques to provide realistic project expectations.

## Estimation Fundamentals

### Estimation Types
- **Effort Estimation:** Person-hours or person-days required
- **Duration Estimation:** Calendar time to complete work
- **Cost Estimation:** Financial resources required
- **Resource Estimation:** Personnel, equipment, and materials needed

### Estimation Accuracy
- **Rough Order of Magnitude (ROM):** -25% to +75% accuracy
- **Budget Estimate:** -10% to +25% accuracy
- **Definitive Estimate:** -5% to +10% accuracy
- **Accuracy Improvement:** Increases as project progresses

## Estimation Techniques

### Analogous Estimation (Top-Down)
- **Historical Data:** Use similar past project data
- **Expert Judgment:** Rely on experienced estimator knowledge
- **Parametric Scaling:** Adjust for project size differences
- **Quick and Cost-Effective:** Suitable for early planning

### Parametric Estimation
- **Statistical Relationships:** Use project parameters for estimation
- **Regression Analysis:** Mathematical models based on historical data
- **Industry Standards:** Use standard productivity metrics
- **Database Support:** Commercial estimation software

### Bottom-Up Estimation
- **Work Breakdown Structure:** Detailed task breakdown
- **Individual Estimates:** Estimate each work package separately
- **Roll-up Process:** Aggregate estimates to total project
- **High Accuracy:** Most detailed and accurate method

### Three-Point Estimation (PERT)
- **Optimistic (O):** Best case scenario
- **Most Likely (M):** Most probable scenario
- **Pessimistic (P):** Worst case scenario
- **Expected Value:** E = (O + 4M + P) / 6
- **Risk Consideration:** Accounts for uncertainty

## Effort Estimation Methods

### Function Point Analysis
- **Function Types:** External inputs, outputs, inquiries, files, interfaces
- **Complexity Factors:** Simple, average, complex weighting
- **Technical Factors:** 14 technical complexity factors
- **Productivity Rates:** Function points per person-month

### Use Case Points
- **Actor Classification:** Simple, average, complex actors
- **Use Case Classification:** Simple, average, complex use cases
- **Technical Factors:** 13 technical complexity factors
- **Environmental Factors:** 8 environmental factors

### Story Points (Agile)
- **Relative Sizing:** Compare story complexity to reference story
- **Fibonacci Sequence:** 1, 2, 3, 5, 8, 13, 21...
- **Team Consensus:** Planning poker technique
- **Velocity Tracking:** Historical delivery rate

## Duration Estimation

### Critical Path Method (CPM)
- **Network Diagrams:** Activity sequencing and dependencies
- **Critical Path:** Longest path through project network
- **Float Calculation:** Slack time for non-critical activities
- **Schedule Compression:** Crashing and fast-tracking techniques

### Gantt Charts
- **Timeline Visualization:** Activities plotted against time
- **Dependencies:** Predecessor and successor relationships
- **Milestones:** Key project checkpoints
- **Resource Allocation:** Resource assignment visualization

### Resource-Leveling
- **Resource Constraints:** Limited resource availability
- **Schedule Adjustment:** Delay non-critical activities
- **Resource Histogram:** Resource usage over time
- **Overallocation Resolution:** Balance resource utilization

## Cost Estimation

### Cost Categories
- **Direct Costs:** Labor, materials, equipment directly attributable
- **Indirect Costs:** Overhead, administration, general expenses
- **Fixed Costs:** Costs that don't vary with activity level
- **Variable Costs:** Costs that change with activity level

### Cost Estimation Methods
- **Analogous Costing:** Use similar project cost data
- **Parametric Costing:** Cost per unit based on parameters
- **Bottom-up Costing:** Detailed cost breakdown
- **Reserve Analysis:** Contingency and management reserves

## Risk and Uncertainty

### Contingency Reserves
- **Known Unknowns:** Identified risks with contingency budgets
- **Unknown Unknowns:** Management reserve for unforeseen issues
- **Reserve Calculation:** Percentage of base estimate
- **Reserve Management:** Controlled release of reserves

### Monte Carlo Simulation
- **Probability Distributions:** Range of possible outcomes
- **Random Sampling:** Multiple simulation iterations
- **Result Distribution:** Probability of cost/schedule outcomes
- **Confidence Intervals:** Range of likely outcomes

## Estimation Best Practices

### Data Collection
- **Historical Database:** Past project data collection
- **Metrics Collection:** Actual vs estimated comparison
- **Lesson Learned:** Project retrospective analysis
- **Industry Benchmarks:** External comparison data

### Expert Involvement
- **Expert Judgment:** Experienced estimator input
- **Delphi Technique:** Anonymous expert opinion gathering
- **Wideband Delphi:** Iterative expert estimation
- **Consensus Building:** Group estimation techniques

### Documentation and Communication
- **Assumption Documentation:** Clear estimation assumptions
- **Range Communication:** Best case, worst case, most likely
- **Confidence Levels:** Estimator confidence in estimates
- **Stakeholder Communication:** Clear explanation of estimates

## Common Estimation Errors

### Optimism Bias
- **Planning Fallacy:** Underestimation of task duration
- **Overconfidence:** Ignoring risk and uncertainty
- **Hofstadter's Law:** Tasks take longer than expected
- **Student Syndrome:** Procrastination due to early deadlines

### Scope Issues
- **Scope Creep:** Uncontrolled requirement changes
- **Gold Plating:** Adding unnecessary features
- **Feature Creep:** Incremental feature additions
- **Requirement Changes:** Late requirement modifications

### External Factors
- **Resource Availability:** Team member availability issues
- **Technology Changes:** New technology learning curves
- **Market Conditions:** External factor impacts
- **Regulatory Changes:** Compliance requirement changes
